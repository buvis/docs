# Catalogue

Configurations and tools managed by buvis/home, grouped by purpose.

## Shell

- **bash** - shell configuration, aliases, functions via bash-it
- **nushell** - alternative shell configuration
- **starship** - cross-shell prompt theme

## Editor

- **nvim** - Neovim configuration

## Terminal

- **alacritty** - GPU-accelerated terminal emulator
- **wezterm** - cross-platform terminal emulator

## Git

- **git** - global git configuration and ignore patterns
- **lazygit** - terminal UI for git
- **gita** - manage multiple git repositories
- **gh** - GitHub CLI configuration

## Dev tools

- **mise** - dev tool version manager (replaces asdf, nvm, pyenv)
- **ruff** - Python linter and formatter
- **bat** - syntax-highlighted cat replacement
- **dircolors** - terminal color scheme for file listings
- **yamllint** - YAML linter
- **harper-ls** - grammar checker language server
- **rumdl** - Markdown linter

## Media

- **beets** - music library manager and tagger
- **smugmug** - photo gallery management

## Applications

- **obsidian** - note-taking and knowledge base
- **vivaldi** - web browser
- **choosy** - browser chooser (macOS)
- **backblaze** - cloud backup
- **fastrawviewer** - RAW photo viewer
- **lightroom** - photo editing
- **snagit** - screen capture
- **syncthing** - file synchronization
- **btt** - BetterTouchTool (macOS)
- **reader** - RSS/read-later
- **nless** - pager

## Finance

- **ynab-buddy** - YNAB transaction import helper
- **fctracker** - financial tracking
- **fadlr** - financial data loader

## Infrastructure

Ansible playbooks in `.playbooks/` for provisioning non-cluster machines:

- **enable-bluetooth-speaker** - configure Bluetooth audio
- **make-music-server** - set up Mopidy music server
- **make-snapclient** - set up Snapcast client for multi-room audio
- **install-docker** - Docker installation
- **upgrade-host** - host OS upgrade automation

## AI

- **claude** - Claude Code CLI configuration, rules, skills, and hooks
- **codex** - OpenAI Codex CLI configuration

## Platform-specific

- **ahk** - AutoHotKey scripts (Windows)
- **powershell** - PowerShell profile (Windows)
- **windows** - Windows-specific configurations
- **wsl** - WSL-specific configurations
- **eObcanka** - Czech eID card tooling
- **shortcuts** - macOS Shortcuts automation
- **tampermonkey** - browser userscripts
