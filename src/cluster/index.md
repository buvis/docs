# Cluster

Documentation for the BUVIS home Kubernetes clusters in a declarative state. [Flux](https://github.com/fluxcd/flux2) watches the [clusters repository](https://github.com/buvis/clusters) and makes the changes to them based on the YAML manifests.

A lot of inspiration for the cluster came from the people that have shared their clusters over at the now-defunct k8s-at-home community.
